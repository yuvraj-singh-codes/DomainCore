# Domain-driven Nest

Applying domain-driven design principles to a Nest.js project. I threw this together rather quick, just to show how each layer operates and behaves.



## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
